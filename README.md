# SampleWebAPI
WebAPI (net 7.0) using the [3musketeers]: https://3musketeers.io/ pattern


# Development Tools:
* dotnet cli
* visual code
* ubuntu 22.04.1 LTS

# CLI Commands:
`dotnet new webapi -o SampleWebAPI`
* root folder:

`dotnet new sln -n SampleWebAPI.sln`
`dotnet sln SampleWebAPI.sln add src/SampleWebAPI/SampleWebAPI.csproj`
